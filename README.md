Prithvi | The Digital Voyager ⛵

Prithvi is a responsive and interactive Digital world explorer application designed to provide some information about all the countries.

🚀 Features:
Dynamic UI: A clean, mobile-first design powered by Tailwind CSS.

Basic Info: flag, capital, language, population, region etc.

Additional info: live weather such as temperature and wind speed, live google map and a cultural note.

⌨️ Tech Stack:
HTML5: Semantic structure for better accessibility.

Tailwind CSS: Utility-first styling for a custom, modern look.

JavaScript (ES6+): DOM manipulation, state management, and logic.

API: Rest Countries API for database and prerequisite of country info, Open-meteo API for live weather such as temperature and wind speed.

Open to all suggestions and critics, also I'm working on a react interface for the same it'll be coming soon.
